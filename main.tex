Let us start with a list, we have

  \itemaccent second item in accent color,
  \item third item in regular color.
\end{itemize}

We can also do enumerated lists.
\begin{enumerate}
  \item Start,
  \item \begin{enumerate}
    \item a subitem,
    \item another one,
    \enumaccent one in accent color,
  \end{enumerate}
  \item the end.
\end{enumerate}

\end{frame}
